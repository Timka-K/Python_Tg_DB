# 📁 Portfolio Manager Bot

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)
[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue)](https://core.telegram.org/bots)

**Portfolio Manager Bot** — твой личный помощник для хранения проектов в Telegram. 📱

## ✨ Возможности

- 📌 __ __Добавляй проекты__ __ — название, описание, ссылка, статус
- 🏷️ **Добавляй навыки** — какие технологии использовал
- 📊 **Отслеживай статус** — в работе, готово, планируется
- 🔍 **Быстрый поиск** — введи имя проекта и получи всю инфу
- ✏️ **Редактирование** — меняй данные в любой момент
- 🗑️ **Удаление** — убирай лишнее

## 🤖 Команды бота

| Команда | Что делает |
|--------|------------|
| `/start` | 👋 Приветствие |
| `/info` | 📋 Список команд |
| `/new_project` | ➕ Новый проект |
| `/skills` | 🏷️ Добавить навык |
| `/projects` | 📂 Все проекты |
| `/delete` | ❌ Удалить проект |
| `/update_projects` | ✏️ Изменить проект |

## 🚀 Запуск за 3 шага

1. 📥 **Скачай** `git clone <ссылка>`
2. ⚙️ **Установи** `pip install pyTeleBotAPI`
3. 🔑 **Настрой** токен в `config.py`
4. ▶️ **Запусти** `python main.py`

## 🛠️ Стек

- Python 3.8+
- pyTelegramBotAPI
- SQLite

---

💡 *Просто отправь название проекта — и бот покажет всю информацию!*
